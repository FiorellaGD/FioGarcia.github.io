# Página Web de Fiorella Andrea García Dávalos

Este repositorio contiene una página web personal desarrollada como parte de un proyecto educativo. En esta página, Fiorella comparte información sobre su experiencia, cursos y otros detalles relevantes.

## Contenido del Proyecto

- **index.html**: El archivo principal de la página web.
- **index.css**: La hoja de estilos que define el diseño y la apariencia de la página.
- **index.js**: Un archivo JavaScript que agrega dinamismo a la página.
- **foto1.jpg** y **perfil.jpg**: Imágenes utilizadas en la página.

## Características

- **Diseño atractivo**: Utilizando HTML y CSS para un diseño limpio y profesional.
- **Interactividad**: JavaScript para un saludo dinámico.
- **Accesible**: Compatible con GitHub Pages.
